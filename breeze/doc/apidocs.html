API Documentation Page

General information

This is RESTful service. All responses are in JSON format. Base url is http://amendia-doctors.360bound.info

Login

Method Url Description
POST /api/users/login

Request:

{ email: E-Mail
, password: Password
}

Response:

  { loginToken: Login token
  , expiresAt: Expiration date/time in unix time format
  }

For invalid credentials you'll get HTTP/1.0 401 Not Authorized

Gettings surgeries

Method Url Description
GET /api/surgeries

Request:

{ loginToken: Login token received in login call
}

Response:

{
  { name: Name of surgery
  , steps: { title
           , videos: {
                       { src: Link to video file
                       , thumbnail: Link to thumbnail image
                       , description: Video description
                       }
                     , { src: Link to video file
                       , thumbnail: Link to thumbnail image
                       , description: Video description
                       }
                     , ...
                     }
           }
  }
}

For invalid token you'll get HTTP/1.0 401 Not Authorized
